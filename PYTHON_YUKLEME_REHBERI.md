# Python Yükleme Rehberi (5 Dakika)

## Adım 1: Python İndir
1. https://www.python.org/downloads/ adresine git
2. "Download Python 3.x.x" butonuna tıkla (en son sürüm)

## Adım 2: Python Kur
1. İndirilen `.exe` dosyasını çalıştır
2. **ÖNEMLİ:** "Add Python to PATH" kutusunu işaretle! ✅
3. "Install Now" butonuna tıkla
4. Kurulum bitene kadar bekle (2-3 dakika)

## Adım 3: Bilgisayarı Yeniden Başlat
- Windows'u yeniden başlat (Python'un PATH'e eklenmesi için gerekli)

## Adım 4: Kontrol Et
1. Windows + R → `cmd` yaz → Enter
2. Şu komutu çalıştır:
   ```
   python --version
   ```
3. Eğer "Python 3.x.x" yazıyorsa başarılı! ✅

## Adım 5: Scripti Çalıştır
1. `CALISTIR.bat` dosyasına çift tıkla
2. Otomatik olarak paketler yüklenecek ve script çalışacak

## Sorun mu var?
- "Python bulunamadı" hatası alıyorsan: Bilgisayarı yeniden başlat
- Hala çalışmıyorsa: Python'u kaldırıp tekrar kur, bu sefer "Add to PATH" seçeneğini işaretle
